# モーニングルーティン

毎朝のルーティンを支援する個人専用のPWAウェブアプリ。  
スマホのホーム画面に追加して、アプリのように使うことを想定しています。

---

## 特徴

- **ログイン不要** - 個人専用、アカウント管理なし
- **完全ローカル動作** - データはブラウザ内にのみ保存（外部通信なし）
- **PWA対応** - ホーム画面に追加してオフラインでも使える
- **モバイルファースト** - 片手・親指操作を前提としたUI

---

## 画面構成

下部タブバーで画面を切り替えます。

| タブ | 役割 |
|------|------|
| ホーム | 今日のダッシュボード（起床時刻・習慣・ルーティン状況） |
| 記事 | 朝活の恩恵・価値についての読み物 |
| タスク | タスクの追加・状態管理（未着手 / 進行中 / 完了） |
| タイマー | 作業タイマー（タスク連携・ゲージ表示） |
| 記録 | 履歴・統計・連続達成日数（ストリーク） |
| 設定 | BGM・通知・データ管理 |

---

## 主な機能

### ホーム
- 今日の日付・時刻の表示
- 起床時刻のワンタップ打刻
- 習慣トラッカー（チェック式）
- 本日のルーティン進行状況
- 連続達成日数（ストリーク）サマリー

### タスク管理
- タスクの追加・削除
- 3段階の進行状態管理（未着手 / 進行中 / 完了）
- 作成日時・更新日時の記録

### ルーティンテンプレート
- 朝のルーティン一式をテンプレートとして保存
- ワンタップで当日のタスク・習慣に一括展開

### タイマー
- 開始・停止・再開・リセット
- 円形または横長の進捗ゲージ
- タスクと連携して計測
- 終了時アラート（音・バイブ・通知）

### 記録・統計
- 日次履歴のカレンダー・一覧表示
- ストリーク（連続達成日数）を主役とした統計グラフ
- データのエクスポート（バックアップ）／インポート（復元）

### BGM・設定
- mp3ファイルのアップロードと管理（IndexedDB保存）
- タイマー・ルーティン中のBGM割り当て
- 通知のリマインダー設定

---

## データについて

> **注意:** データはこのブラウザ内にのみ保存されます。  
> ブラウザの「サイトデータ削除」や端末初期化を行うと記録が失われます。  
> 定期的に「設定 → データエクスポート」でバックアップを取ることを推奨します。

### 保存場所

| データ種別 | 保存先 |
|-----------|--------|
| 設定・タスク・習慣・ログなど | localStorage |
| 音声ファイル（mp3） | IndexedDB |

### エクスポート形式
音声以外のすべてのデータをJSONファイルとして書き出します。

---

## 通知について

通知はベストエフォートでの実装です。  
アプリが起動中・または直近に使用した状態でのみ確実に動作します。  
iOS Safariでは、アプリを長時間完全に閉じた状態での予約通知は保証できません。

---

## デザインコンセプト

- **テーマカラー:** ミント系パステル
- 朝らしい柔らかさと清潔感を意識したUI
- タブ切り替えや完了時のマイクロインタラクションで達成感を演出

---

## 開発・実行方法

バックエンドは不要です。ローカルサーバーで `index.html` を開くだけで動作します。

```bash
# Python 3
python -m http.server 3000

# Node.js (npx)
npx serve .

# VS Code
# Live Server 拡張機能でindex.htmlを開く
```

ブラウザで `http://localhost:3000` を開いてください。

---

## スコープ外（このアプリで提供しないもの）

- 複数ユーザー・ログイン機能
- 端末間のデータ同期
- クラウドバックアップ
- 外部サービス連携（カレンダー・天気APIなど）
